    Hugo adalah static generator tercepat di dunia yang mempu membuat ribuan postingan hanya dengan hitungan detik.

    Install Hugo Di Windows

    Cara install hugo itu mudah dan tidak membutuhkan software pihak ke-3 lainnya seperti (Node.js). Anda hanya perlu memasangnya dengan cara mendownload hugo versi terbaru disini lalu ekstraks hasil download pada folder C:\Hugo\bin dan rubah nama menjadi hugo.exe jika bawaannya sudah hugo.exe maka tidak perlu ada perubahan lagi.

    Setelah proses ekstraksi selesai, selanjutnya Anda perlu mengatur Envirnoment Variables dan menambahkan ;C:\Hugo\bin caranya

    1. Klik Tombol Windows > Sistem
    2. Klik Advanced systems settings
    3. Klik tombol Environment Variables…
    4. Klik PATH/Path lalu edit
    5. Masukan Direktori Hugo ke dalam Path dengan tambahan tanda “;” tanpa kutip seperti ini: ;C:\Hugo\bin atau bisa Anda lihat pada gambar yang ditandai biru. Jangan lupa untuk menekan tombol Apply dan OK

    Untuk pengguna Windows 7 dan 8.x:

    Khusus untuk Windows 7 dan 8.1 tidak menyertakan editor untuk Environment Variables yang disertakan di Windows 10, jadi pengguna non-teknis di platform tersebut disarankan untuk memasang editor gratis yang dibuat oleh pihak ketiga seperti Windows Environment Variables Editor atau Path Editor.

    Sekarang Hugo telah terpasang. Selanjutnya, Anda dapat membuat situs pertama dengan static site generator Hugo.
